1. Abstarct:

DAVE is a web application developed by students. The purpose of this project is to improve the security of data transferred and stored in on-line environment.

This report will present the details behind the development of this project and its progress.

2. Introduction:

Nowadays every web-site process its user's data, no matter if it's about transfering or storing it and inevitable the security of this data is one of the current issues that people are dealing with. DAVE is projected for every website owner who wants his users data to be stored in a secured way. Also, he will be able to visualize this data both in a raw format and also in a processed one (displaying different relevant statistics).

3. Goals:

  • functional platform where you can manage your websites forms security
  • possibility to visualize the data submitted through your websites forms
  • get relevant statistics about this data

5. User experience:

  • At start-up, the user will have the option to login if he is a member of DAVE Community, or to register if he isn't.
  • After login he will be redirected to "Home" page. Here he will be able to read a description about the application, to read frequently asked question or to get in touch with the developers. He will also be able to acces different pages within the platform: "Keys Library", "Data Visualization", "Profile".
  • When the user will acces "Keys Library" page he will be able to manage his associated keys and to generate new ones.
  • When the user will acces "Data Visualization" page he will be able to visualize his forms data and to filter it by different methods.
